Perfetto UI (2019):

This the project that I worked on last year during my internship at Google. This project was open source so that usedul when people asked me what I did at Google. Perfetto is used to understand what is heppeneing when apps are opened on phones and help analyze crashes or how much of the phones functions an app uses. For this project we created a way for the screen of the phone to be recorded along with the insides of the phone to see what the user sees. You can see the website at Perfetto.
